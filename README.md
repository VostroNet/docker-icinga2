# supported Environment Vars

for MySQL Support:

  - MYSQL_HOST
  - MYSQL_PORT
  - MYSQL_NAME
  - MYSQL_ROOT_USER
  - MYSQL_ROOT_PASS
  - IDO_PASSWORD

for graphite Support:

  - CARBON_HOST
  - CARBON_PORT

for dashing Support:

  - DASHING_API_USER
  - DASHING_API_PASS
